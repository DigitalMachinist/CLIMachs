Class: CommandPermissions

.cli. CommandPermissions

CommandPermissions objects are used to define a whitelist of conditions under which a message is allowed to execute a command belonging to the CLI.

groups :CLIMachs.collections.UniqueCollection

A list of player IDs that represent players who have been granted full-time exclusive access to this command. This kind of permission granting should be reserved for co-GMs or helpers that need higher-level access to CLI commands than players do. Note: This getter returns a shallow copy of the data array, so operating on it will not affect the collection.
