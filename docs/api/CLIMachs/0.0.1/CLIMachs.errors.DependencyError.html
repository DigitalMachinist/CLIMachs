Class: DependencyError

.errors. DependencyError

DependencyError is a subclass of Error for semantic purposes so denote that an action could not be completed because some other resource was expecting something to exist or otherwise not change. It doesn't do anything new, it's just to distinguish error types.

new DependencyError(message)

Parameters:
Name Type Description
message string

A short description of the error.
